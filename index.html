Autonomous Rover

Project Overview

This project involved the complete design, fabrication, and programming of an autonomous rover for simulated search and rescue missions. The primary objective was to create a robust platform capable of navigating complex, unknown environments using sensor fusion and pathfinding algorithms. The rover successfully demonstrated its ability to map its surroundings, identify target objects using computer vision, and transmit data back to a base station in real-time.

Key Technologies

  • Raspberry Pi 4
  • Python (OpenCV, NumPy)
  • LiDAR Sensor
  • ROS (Robot Operating System)
  • 3D-Printed Chassis (Fusion 360)
